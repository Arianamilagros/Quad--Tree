# Quad--Tree
QuadTree aplicado a imagenes como caricatura y paisajes.

Python 2.7.8

Librerias graficas:
Numpy 1.8.2
Opencv 2.4.9

Compilar:
$ python pyImgQuadTree.py pato.jpg -sz 20 -std 25.0
